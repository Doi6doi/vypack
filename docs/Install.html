Install

Compilation and install instructions for vypack.

Compilation

Run vymake -f https://raw.githubusercontent.com/Doi6doi/vypack/main/Make.vy.
It will clone the repository and build the library.
If you already have it cloned you only need to run vymake without arguments
Tools needed to build:
-vymake
-a vymake compatible C compiler (gcc, clang, msvc, ...)
-a vymake compatible git client (only for cloning)

Installation

You can run the resulting binary from any location.

Debian package

On a Debian-compatible linux, you can also create a debian package (.deb)
Run vymake deb in project directory
Tools needed:
-vymake
-dpkg-deb
